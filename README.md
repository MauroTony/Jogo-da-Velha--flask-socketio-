## Instalação

```bash
# Clone o repositorio:
git clone https://github.com/MauroTony/Jogo-da-Velha--flask-socketio-.git

# Entre na pasta
cd Jogo-da-Velha--flask-socketio-

# Instale as dependências
pip install -r requirements.txt

# Execute o servidor
python app.py
```

Accesse em <http://127.0.0.1:5000/>
